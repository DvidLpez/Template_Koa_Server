# Template Koa Server

Template to create an API with endpoints REST and GraphQL

## :dart: Features

 - Endpoints REST
 - Endpoint GraphQL
 - Connect with other APIS
 - Mongo Database
 - MySQL Database

## :computer: Tech Stack

 - [Koa](https://koajs.com/)
 - [GraphQL](https://www.apollographql.com/)
 - [Typescript](https://www.typescriptlang.org/)
 - [TypeORM](https://typeorm.io/)
 - [Apollo](https://www.apollographql.com/docs/apollo-server/integrations/middleware/#apollo-server-koa)

## :gear: Installation
- NPM depencencies and devDependencies, see package.json


## :eyes: How to usage

### Docker Compose

``` bash
#Up service
docker-compose up -d mongo
# Down service
docker-compose down
```
  
### Start aplication
``` bash
yarn run dev
```
